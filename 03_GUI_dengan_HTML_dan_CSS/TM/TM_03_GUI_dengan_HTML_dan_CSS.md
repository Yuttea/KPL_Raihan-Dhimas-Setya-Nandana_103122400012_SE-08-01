## Tugas

terapkanlah fungsi untuk (1) menghitung huruf kecil yang disediakan di #hk, (2) mengubah huruf kecil ke huruf besar ketika pengguna menekan tombol #huruf-besar, dan (3) mengubah huruf besar ke huruf kecil ketika pengguna menekan tombol #huruf-kecil.

Untuk nomor 2 dan 3, tampilkan hasilnya di dalam editor-kecil.

Kemudian, hapuslah fitur "Paragrafkan" dari alat.

## Deskripsi

Program tampilan awal menampilkan web Pengkorversian Gaya Teks setelah menghapus fitur/tombol "paragrafkan".  
Pada bagian menghitung huruf kecil, akan menampilkan jumlah huruf kecil saat mengetik. Misalkan mengetik "Rafael Putra Septava", maka huruf kecil yang dihitung adalah 15.  
Pada bagian mengubah huruf menjadi besar, setelah menekan tombol "Besarkan", maka akan mengubah huruf kecil menjadi huruf besar, sedangkan yang awalnya huruf besar tidak berubah.  
Pada bagian mengubah huruf menjadi kecil, setelah menekan tombol "Kecilkan", maka akan mengubah huruf besar menjadi huruf kecil, sedangkan yang awalnya huruf kecil tidak berubah.
